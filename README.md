# 🚀 Простой Мессенджер

Быстрый real-time чат с Socket.IO, готовый для деплоя.

## ⚡ Быстрый запуск

```bash
npm install
npm start
```

## 🌟 Особенности

- ✅ **Real-time** - мгновенные сообщения через Socket.IO
- ✅ **Простой интерфейс** - красивый UI без регистрации
- ✅ **Мобильная версия** - адаптивный дизайн
- ✅ **Готов к деплою** - один файл, все включено
- ✅ **Бесплатный хостинг** - совместим с Railway, Vercel, Render

## 🔧 Технологии

- **Backend**: Node.js + Express + Socket.IO
- **Frontend**: React (без сборки)
- **Стили**: Pure CSS с анимациями

## 🌐 Деплой на Railway

1. Зайти на [railway.app](https://railway.app)
2. "Deploy from GitHub repo"
3. Выбрать этот репозиторий
4. Готово! ✨
